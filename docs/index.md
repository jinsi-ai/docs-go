## 什么是 DocsGo？

**DocsGo** 是一款基于 **Go语言** 开发的开源 **Markdown文档服务器**，专为技术团队、开发者和企业设计。

与传统静态网站生成器不同，DocsGo采用**实时渲染**架构，让你**无需编译**即可即时预览文档更新。

### 为什么选择 DocsGo？

- ⚡ **实时预览** - 修改即刷新，无需重新编译
- 🔍 **全文搜索** - 基于SQLite FTS的即时搜索
- 🔒 **密码保护** - 站点级和文档级双重加密
- 📦 **单文件部署** - 一个二进制文件，零依赖
- 🌍 **国际化** - 完整中英文文档支持

## 快速开始

```bash
# 下载并运行
wget https://github.com/jinsi-ai/docs-go/releases/latest/download/docs-go-linux-amd64
chmod +x docs-go-linux-amd64
./docs-go-linux-amd64

# 访问 http://localhost:8080
```

更多安装方式请查看 [快速入门指南](./zh/quickstart)
